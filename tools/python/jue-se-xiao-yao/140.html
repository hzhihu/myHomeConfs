第四卷 凤皇于飞 第一百三十九章 降服战神

陪着这个金海凌应酬了一天，直到晚宴结束，我才大大地松了一口气。

我从来没有碰到过这么会粘人、而且这么会说话的男人。就算是焰，也不及他的一半。

他的身上，有着一种北方男人特有的豪爽，不同于温润的雪月国人，但说起话来，又是咄咄逼人，常常让我觉得他非常难应付，总是要我脑筋高速转动，才能对付得了他。

结果这一晚上的唇枪舌战下来，不但没有让他对我的兴趣减少，反而那双看我的眼睛，更是明亮了不少，直到最后，他还摆出一副依依不舍的样子，无奈地回房。

我甩手捶了捶坐了一天有些酸痛的腰，真是要命，应付他一个人，比应付平常的十个人还要累。我衷心的希望，他能够赶紧离开雪月国！

我有预感，只要这金海凌一天不走，我就有得烦了！

果然，第二天，我还没睡醒，小蝶就小心翼翼地进来禀报，说凌亲王在外面求见。

我恼怒地咒骂了几声，平常不上朝，我是一定要睡懒觉的，现在正睡得舒服香甜的时候，他就跑来闹了，真是个烦人精！呆会不给他好脸色看。

在小蝶的侍候下，我快速地洗涮好，换上一套以前自己亲自设计的淡紫色宫纱裙。我喜欢紫色的神秘，穿上身整个人美得如梦如幻，连我自己都对这套纱裙爱不释手。

整妆完毕后，在小蝶的陪伴下，我才缓缓地走了出门。

当我浅笑着立在金海凌的面前时，他的眼里闪过一阵阵的惊艳，一双明眸死死地紧盯着我，竟就这样看失了神。

直到我忍不住掩嘴轻咳了一声，他这才回过神来。

但却丝毫不因为自己的失态而显得忸怩，平时的人在我轻咳一声，必定是脸红心跳。可是这个金海凌呢，他很快便稳住了神，反而直接笑着开口赞美起来，“女皇陛下真是魅力无边，清丽绝俗，冰清玉洁，有如画中天仙，堪比神妃仙子，真是让本王看呆了眼，丢失了魂啊。请陛下原谅本王的失礼！实在是因陛下太美了！让本王忍俊不住，现在，本王才知道什么叫做倾城倾国呀！在本王看来，这倾城倾国也不足以形容女皇陛下的动人之美啊！”

这家伙说起好话来，倒是一撂一撂的，这伸手不打笑面人，好话谁不喜欢听？他的话，确实让我刚刚的起床气消了一大半。

我朝他扬起一抹轻笑，“王爷的这张巧嘴，可真会讨女人的欢心哪！不知王爷如此早就过来找朕，是有什么急事呢？”

我话里暗喻的就是，没有急事的话，你就不应该现在跑来扰我清梦。

“女皇陛下，你看，今日太阳高照，风和日丽，如此良辰美景，不出来活动一下，岂不是浪费了好时光。自从那日在战场上，本王败于陛下之手后，本王回国后又拜遍名师，潜心苦练，最近也略有小成。本王一向是个武痴，一直希望能有机会再与陛下切磋一二，今天既已到此，故特希望陛下能一偿本王所愿。不知陛下可否应承本王的这个无礼之请？”

金海凌语气虽然诚恳，但是眉宇之间的傲然，却丝毫没有收敛，甚至眼里还闪着一丝想彻底驯服我的迫切。

我的眼珠轻转，明眸闪过一丝狡黠和算计，对着金海凌巧笑盎然，“王爷看来确是武功大进了，可是，朕却最近偷懒了，一进一退，朕怕已不是王爷的对手啊！到时候，万一伤了王爷，那可就伤了和气了。”

“呵呵，陛下过虑了！本王邀请陛下切磋，可不是为了伤和气的。双方点到为止，陛下，你说这样可好？”金海凌那双闪亮的星眸，竟然又朝我发出一股超强的电力，慑人心魂的电眼，猝不及防之下，差点被他电晕了我。

汗……这个男人，看来随时随地都在想着办法来勾引我。

我装作没有感觉到他发出的无声邀请，仍然是谈笑自若地说，“当然可以！那就这么说定了，点到为止！”

果然，看我对他的勾引没有什么反应，金海凌的眼里闪过一丝失望。

“小蝶，去请赵都统和程尚书到御花园东面的草坪上，朕和王爷要在那里切磋，请他们当个见证人。”我对小蝶说。看着小蝶领命而去后，我又回过头笑着对金海凌说，“王爷，请随朕来吧！”

*****

御花园东面草坪上，很快便围聚了几个人。

我和金海凌两人相对而立，边上三米远处，站立着赵毅和程知礼、小蝶和小舞，另外还有一位就是金海凌的贴身侍卫。而周边时的花从中，时不时有身影快速晃过，想来便是那些不想错过此等比武盛事，却又未得我征召前来的人在偷偷地观看。

为了怕误伤，在我的建议下，我和金海凌赤手空拳上阵。

“王爷，请！”我朝着金海凌笑得甜甜的，心底里却在算计着，呆会该怎么挫他的锐气。

“陛下，先请！”金海凌很有风度的拱手请我先出手。

“那朕就不客气了！”话音一落，我便使出综缈步法，身形灵动如轻鸟一般地，手掌轻扬，漫天的逍遥掌影，便朝着金海凌的身上罩了过去。

我还故意留了三成功力，估计七成功力足已对付他了。

这金海凌倒是临危不惧，脸上一副坦然，一个闪身，不但轻易突破了我的掌影包围，反而还顺势地一掌朝我打了过来。

呵呵……看来这小子还真是长进了不少啊！

虽然功力高于他，但我也不敢再掉以轻心，怕万一被他来个暗手，赢了我，那岂不是没整到他，还让自己丢脸，多不划算啊！

就这样一来一往，彼此也打了近百招。

而边上的众人，只感觉场中一紫一白的两个身影，有如飞龙在天，不停地在那里飞来闪去。又有如两个人在跳着完美的双人舞，流离舞动，互相缠绕，像是表演，甚是好看。

只听程知礼在那里一边看一边说，“赵都统，在下可真是从来没见过，比武也能比得如此好看、如此协调的。真是奇也怪哉！”

赵毅的娃娃脸上闪着一种与我荣焉的自豪感，带着一种膜拜的向往，看着场中那抹紫色的身影说，“那是！这可是我们艳绝天下、武功盖世的女皇陛下，她的风采，怕是这个天下，亦无人能出其左右啊。”

“赵都统说的是！陛下宽厚待人，废除了不少往时的陋习，积极改革新政，她做的每一件事，无一不是让老百姓受惠、让雪月国未来愈来愈好的。在下纵然粉身碎骨，赴汤蹈火，也要追随在女皇陛下的左右，为其效犬马之劳！能跟在女皇陛下的身边效劳，真使程某的脸上感到无比光荣啊。咱雪月国有了女皇陛下，确实是雪月之幸，万民之福啊！”

“是！是！是！”赵毅连连称是，“程尚书说的，正是在下想说的话。”

他们两个交谈的话，一句不漏的落到了我的耳里，一种欣慰之感油然而起，能得到他们这样的称赞，那就证明，我所做的一切，都没有白费了。

其实，在这些荣誉的后面，更多的是焰他们的功劳。因为，他们才是真正的计划执行者。而从现代经理人的管理经验来看，我深深知道，一个新的计划，要在整个公司执行下去，就有不小的阻力和困难，更别提在一个国家实行新政了，如果没有强大的执行力，一切等于只是一场空话，没有任何意义。

想到我亲爱的夫君焰，我的心神又渐渐地恍惚起来。

失神之间，竟差点就被金海凌一招得了手，感觉到掌风从我耳侧掠过，虽然避了过去，但却险些落了下风。

我赶紧收摄起心神，全心应付起他来，掌上也不由得加了几分力道，不想再跟他耗下去了。

来来回回之间，终于被我逮到了一个机会，我一个闪身转到他的身后，掌风击在他的后背，顺手在颈处轻拂了一下，顿时让他四肢发软地直接摔了出去，摔了个狗啃泥后，趴着摊在地下动也不动了。

看到他的狼狈，我真想大笑几声。我就是故意让他出丑的，谁叫他那么嚣张，不吃吃苦头，他还不知道天外有天，人外有人呢！

看到大家低着头闷笑又不敢出声，我走了过去，低下头看着金海凌的俊脸，此时早已经黑成一团，眼睛闪着一抹浓烈的戾气，看了让我心里一惊，他果然还是那个狠厉的家伙，真是狗改不了吃屎了，轻轻一招，就让他露出了真面目。

不过，这也是难怪他的，堂堂一个王爷，来到我国出使，我身为女皇陛下，不但没有帮他立威，反而亲自来教训他，从大道理上来讲，我是站不住脚的，只是小女人心态，纯粹泄个私愤而已，但是，落在有心人的眼里，到时，就可以大做周章了。

趁着他还未发飙，我赶紧亲自把他扶了起来，还亲切地帮他拍了拍衣服上的草屑，绽放出一脸饱含着歉意的笑容，朝他轻柔细语地说着，“王爷，真是抱歉！朕出手太重，竟然导致王爷摔了一跤，都是朕的错，朕在这里向王爷赔不是了。”

说完，我朝他倾身福了福，带着笑颜，用一双盈盈秋水的明眸轻瞅着他，我就不信，我摆出这样一副低姿态，他还不见好就收，趁机下台。
