第一卷 烟雨濛濛 第二十二章 战神冥向天

“这个……那个……”我嗫嚅了半天，没有说出话来。

“什么这个那个的，快说！”他暴怒了。

死臭小子，你凶什么？要不是老娘我救了你，你早就到阴间去吱歪了，还牛什么牛？我心底冷哼一声。动作却慢条斯理地把俏脸凑了上去，对上他寒光闪闪的双眸，故意娇滴滴地说，“公子，你要奴家说什么好呀？奴家既没有高深的武功，又没有可以吓死人的势力，你说，奴家一个小女子，能拿那些人怎么办？恐怕……到时奴家连自个也保不住了。”说完，我一脸害怕、满脸哀怨地低垂下了头。

我偷偷地从垂下的眼帘底下偷视了他一眼，借着那矇眬的光，竟然看到他刚硬冷酷的脸上有了一种难得的柔情和疼惜，哈哈哈……偶在心底下狂笑得差点抽搐，原来我的演技还有这么好，竟还能骗到这个暴男。

“别怕，有我在！我会保护你！”他很阿沙力的保证，伸出手稍稍地用力一拉，我便站立不稳落在了他的怀里。鼻翼间还满是那浓浓的药草味，混合着一种陌生的男人气味，闯入我的鼻间，配上那有点笨拙、有点动人的话语，竟也薰得我脑子发晕。

这时，外面杂乱的脚步越来越近了，他还是搂着我不肯放。我挣扎了一下，低声说，“放开我！我去看看！”

“不用看了！对方来的人一共有十个，只有一个武功高一点的，但都不是我的对手。所以，这些人没有威胁性，你就安心吧！”他一口气说完，大手还在我的后背摩挲了几下，害我硬生生的抖了两下。

这家伙有那么厉害？这人还没到，他就已经从脚步声里听出来了？看来他也是个武功超高的人哪！那伤他的人武功不是更高？或者他是不小心才被人暗算了？

杂乱的脚步声停在了门口，一声熟悉的带着焦急的声音响了起来，如同雷击在我身上，“雪儿，雪儿，你在吗？”

我的身子颤了一下，二哥，是二哥来找我了！我的心激动了起来，呼吸也跟着紧了。我想推开他，马上起身去见我的二哥，经过了这么半天的折腾，我也想明白了，既然已经错了，就要勇敢承担错误的结果，逃避现实是没有用的，也不是我的风格。

可是，揽住我的这位大哥却丝毫没有放开我的意思。

“是我二哥来找我了！”我不知道为什么要跟他解释。

还好，听到我的话，他倒也没有啰嗦，在我发间深吸了一口气，便马上就放开了我。

一脱离开他的怀抱，我便马上跳下床，朝门边奔了过去，心，竟然很急，也很雀跃。

用力地拉开门，门外站立着我心心念念的他---我的二哥。在明亮火光的照耀下，他全身蒙上了一层淡淡地金光，如同黑夜星空中那亮眼闪烁的星星一般，泛着夺目的光彩，只是这个星星的脸上，现在有着疲惫不堪、却又欣慰的神色，让我的心硬生生的痛了一下。是我！是我让二哥受累了，顿时一股酸涩从心里涌起，眼里瞬间盈满了心疼的泪水。

“雪儿……雪儿……二哥终于找到你了！”风沁阳满脸激动地大步向前，狠狠地将我拥入怀里紧紧地抱着，良久，良久，仍是不肯放手！

我珍惜地回拥着他，舒适的窝在他的怀里，就像以前一样，温暖安全，让我感觉天下的事有他替我扛着。

突然，我感到背后像是有一箭寒光，像要射穿我似的充满杀气，让我从心里感到毛骨悚然。我赶紧在二哥的耳边低语了一句，“二哥，这还有人呢！快放开我！”

风沁阳像是这时才想起周围还有人呢，也识趣地赶紧放开了我，“雪儿，快跟二哥回去吧！”

我仰头望着这个在我心中有如天神一般高大的男人，我不能再增加他的负担了，他的心里并不比我好过多少，朝他浅笑着点了点头，“二哥，我跟你回家！”

他愣愣地看了我一会，眼里有着浓浓地爱恋，半晌才叹了一口气，牵起我的手，“走吧！回家！”

才刚走几步，我才又想起屋里的那个男人，赶忙停下脚步，“二哥，等一下。里面还有一个人，是我昨晚救回来的人。”

“救人？你救人了？我看看！”二哥放开我的手，大步踏入房里面。

在明亮的火把照耀下，我终于可以清清楚楚地看清他的脸了。这会他正侧着用右手撑起半个身子，冷眼挑眉的看着从门外进来的我俩，那双眼睛，是单眼皮的凤眼，微微上挑着，有着傲然，也有着我想像中的寒光四射，薄唇紧抿着，整张酷脸看起来更是像冰块一样，让人感觉森冷异常。虽然一身是伤，却仍然显出一种足以致命的慵懒，如优雅的非洲豹一样，散发出迷人的神采。

“你是……冥将军？？？”二哥带着迟疑的问话，让我的心震了震，有点不敢置信的多看了他几眼，正对上他带着探视的目光看着我。

原来他就是威震澜沧的有着“战神”之称的冥向天大将军，呵呵，还真是想不到，我随手一救，就救了一个大将军哩！说不定以后还能利用利用这个难得的资源呢！我的眼里闪着算计的光芒，脑海里不由自主的又浮现出他那有着完美比例的魔鬼身材，和刚才那个有点温暖的怀抱，我感到血往头上涌，怕他看见我的羞窘，赶紧低下了头。

我听到了他沉稳有力的回答，“正是！不知阁下是哪家府里的公子？”怎么我感觉背心有冷嗖嗖的感觉啊？
