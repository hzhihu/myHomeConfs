第三卷 凤飞雪月 第一百二十二章 打就打，谁怕谁

听到赵毅不顾身份的说话，陆青的脸上马上恢复了平日的邪魅狂放，他鹰眸轻眯，刹时寒气四溢，傲然不惧赵毅身上的怒气，“赵都统，虽然你的官阶是比陆某高，但这是下官和皇上之间的私事，这好像还轮不到赵都统您来插嘴吧？”顿了顿，他又摆出一副了然的表情，“哦，我明白了！赵都统你这么生气，难不成你也和陆某一样，暗里喜欢皇上？”

陆青话里连讥带讽，把个赵毅气得那脸是红一阵白一阵，咬牙切齿却又该死的恰好被陆青说中了他的心事，憋屈着，恼怒着却又说不出话来。

最后，憋了一肚子气的他，恼羞成怒，扬手一拳就朝陆青坏笑的脸上挥了过去。

陆青冷冷一笑，身形轻轻一避就避过了赵毅的拳头。看着赵毅下一拳又挥了过来，他也毫不客气地迎了上去，摆出一副你欠揍的姿态跟赵毅拼了起来。

打就打，谁怕谁呀？

看着两个扭打在一起的男人，我就这样瞪大眼睛傻站在一边，这是什么跟什么呀？这俩男人，是神经哪里脱线了？

看着两个大男人你来我往的斗着，我摇头轻叹，要打，你们就打个饱吧！我可是饿极了！

随手系好身上的白衣，又瞄了一眼地下刚刚被赵毅摔落一地的饭菜，我苦笑着摇摇头，这俩男人，只顾争风吃醋，完全不管我这个皇上的死活，唉！比起事事以我为先的焰来，还真是差远了！

呜……无奈，自己动手，丰衣足食，我还是自己去找吧！想罢，我扬起身形，便往门外飘去，我要去找吃的，来祭我的五脏庙！

打了半天的两个男人，一转眼才发现女主角都已经不见了，这还了得？不约而同地收了掌，又像小孩子似的朝对方冷哼一声，这才争先恐后地朝门口闪去。结果旗鼓相当的俩人，在门口又撞到了一起，互瞪两眼，又抢着往外冲，急着追向前方身形快渺的佳人去也！

看着后面追上来的两男人，我冷瞥了一眼，凉凉地说，“你们打完了？”

赵毅憨厚地嘿嘿一笑，“皇上，微臣和陆都察只是切磋一下武艺，切磋一下而已，没吓着皇上吧？”

陆青则冷哼一声，一脸狂傲地说，“皇上，最近微臣手有些痒，忍不住就想找欠揍的人揍几下挠一挠。”

老实的赵毅又给他嚣张的话给气糊涂了，举手又想打，我轻轻按住他的手，朝他们冷哼一声，“你们有完没完？烦不烦？都给我滚回去，不准跟着我！”

“皇上……皇上还是回屋吧，微臣再差人做好饭菜，就给皇上送过去。”还是赵毅比较懂事，晓得我的需要。

我点了点头，“好！赵毅，你先去吩咐厨房做饭。陆青，你陪我去看看万斗去。”

赵毅看着边上一脸得色的陆青，瞪了他一眼，才带着一丝恼恨地应了声，“是！”

看着老实的赵毅一脸郁闷的离开，我带着嗔怪地看着陆青，“陆青，赵毅是个老实人，你不要欺负他。”

陆青被我这么一责怪，俊脸上闪过一丝受伤的痕迹，但随即便低下头应了声，“是！”

看着赵毅愤然离开，再看看陆青一脸无辜受伤的表情，我的心里也像是被压了块石头似的，沉闷得压得我快要窒息。

我这是招谁惹谁了？（某懒：你招陆青了，所以才会惹到赵毅！某雪：呜……我闪！）

*****

当杜鹃城主万斗看见我驾临他的住处时，脸上并无太大的惊愕和意外。可能是有人已通知过他，他也一副随时恭候的模样了。

一见我进去，赶紧下床给我磕头，“微臣万斗参见女皇陛下，皇上万岁万岁万万岁！”

我伸手扶起他，“万城主有伤在身，就不用多礼了，还是回床上躺好吧！”

“皇上仁慈，果然如传言之中所说一样。微臣佩服！微臣的伤不碍事，还是站着好！”

我仔细打量着这个坚持原则的万斗，只见他面若秋月，儒雅斯文，举止适度有礼，眉宇之间却又有淡淡的倔强，难怪如此坚持原则。嗯！给人的第一印象还蛮不错！

“嗯。万城主，朕很好奇，你为什么要举办这个招亲大会呢？”我看着万斗，带着兴意问道。

“回皇上，这个杜老爷自我上任之后，曾多次拉笼微臣，想让微臣利用职权为其大行方便，也曾托人几次上门，要将杜家小姐许配于我，均被微臣拒绝了。后来，他便用武力逼微臣就范，情急之下，微臣只有想出这个招亲比赛的办法来应付杜老爷。却未想到，这个杜家小姐竟然琴棋书画一样不落，还真给她赢了头彩。这下微臣再也无话可说，只好娶了她！不过，微臣是另有打算，既然他们执意要拉笼我，还如此的费心。且微臣知道他们在暗中拉笼了不少的官员为他们效力，所以，微臣想着，既然躲不过，那就将计就计，不入虎穴，焉得虎子。正好也可以查探查探他们到底暗中在干什么？只是微臣却未想到杜家父女竟然如此大胆，敢以下犯上，犯下滔天大罪，请皇上恕微臣愚昧之罪！”万斗说完，直挺挺的跪在我的面前。

“爱卿何罪之有？好！朕免你的罪。”这个万斗行事原则就像个书呆子，好在除了书呆外，还有一些玲珑。如果我不这样说，他肯定不肯起身吧？

“只是……”我带着冷意的一声只是，又让万斗跪了下来，“万城主为杜鹃城一城之主，杜家的活动如此猖獗，却为何未上报朝廷？”

“回皇上，微臣曾三次递谏上京，却不知为何一直未有消息。”万斗恭敬中带着一丝疑惑。

“陆青，你记一下，到时查一查这件事，万城主上谏的这些卷宗到底到了谁的手中？”

“是！”

“好了！万城主，朕就把杜鹃城托付给你了，希望万城主能不负朕望，好好管理杜鹃城。”我完全的信任他，其实不信任也不行，待回京以后，重订了官员制度后再作打算。

“哦，对了！万城主，吏部下发的六月公考的事，这边反应怎么样？”我突然想起这件事来，还不知道老百姓们怎么想的？

“回皇上，老百姓是一片欢腾啊！各书生都希望能有机会一展长才呢，这回皇上可是提供一个大好机会给他们了，他们都说，终于有出头之日，有了盼头了。臣代书生们谢谢皇上。”万斗一脸欣慰、带着感慨地说。

“好！那就好！”我也高兴自己这个提议可以让更多的人有机会，“不过……这考试防的就是作弊！万城主，你可要看好了，千万不能出现试题泄露之事啊！现在试卷尚未发下，你们下面的衙门都作好准备没有？”

“微臣都安排好了！就等着6月15的到来了。”

“启禀皇上，饭菜已经做好了！”此时，赵毅进来了。

太好了！终于有饭吃了。“好！那有事万爱卿再启奏吧！朕就先回房了。你好好休息！养好身子了，才能有好精神处理公事啊！身体可是做事的本钱哪！”呵呵，怕革命他听不懂，改为做事了。

“谢皇上关心！微臣恭送皇上！”万斗这人，还真是一板一眼的。

*****

回到房中，我大快朵颐，痛痛快快地吃了个饱，那狼吞虎咽的模样，让站在身侧的两个男人目瞪口呆。

一放下筷子，赵毅便乖巧地递上一杯茶，他跟了我一段时间，倒是挺了解我的习惯。

无意中一瞥，又看见两个男人在吹胡子瞪眼的，我淡淡地说，“你们都下去吧！朕要休息了！哦，对了，陆青，杜家老爷的案子可以收尾了吗？”

“基本上可以了！”

“你抓紧时间处理一下，朕想明天启程回宫。”我突然有一种归心似箭的感觉，想回宫了，一个星期了，好想我的小辰星，还有小夜，我想回去了！

“是！那微臣去安排行程！”陆青应允。

第二天，刚吃完早膳，听完了陆青的结案奏折，拜别了万斗，刚准备启程，就收到了宫里皇叔传来的六百里加急快报，说轩辕翌后天就要到雪月国出访。

这轩辕翌搞什么名堂？怎么焰前脚一走，他后脚就出国跑到我这来了？真是诡异之极！不知道这腹黑男又要玩什么花样？

不过，他以一国之君的名来出访，我这个一国之君倒真是不能避的，回宫就回宫吧！看看他耍什么花样？希望这个腹黑男不会给我出太大的难题！
