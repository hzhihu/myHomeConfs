第四卷 凤皇于飞 第一百四十三章 又有了？谁的？

太医馆的门口，站满了人，大家的脸上都挂满了担心和害怕。

到了这个时候，看到胡太医已经开始着手治闻，我这才冷静了些许。转头对着一起跟了过来的程知礼说，“程尚书，你去看一下安南国的王爷，帮朕安顿好他。就说朕恭喜他赢了，其它的一切，等帝君醒了再说吧！”

“是。微臣马上就去。”

程知礼走后，我又对其它人说，“你们不要都堆在这里了，你们也帮不上忙，该干嘛的，干嘛去！”

众人闻言，应了声“是”后，便火速地退下。

太医馆门口很快就只剩下我、冥向天、阿飞还有轩辕凛四个人，看着半天这胡太医还没出来，我的心又吊了起来，无助地抬头望向冥向天，紧张地又问了一句，“向天，这胡太医搞什么，怎么到现在还没出来？”

“你别着急，二弟是不会有事的。淡定一点。”冥向天温柔地安慰着。

我感觉到有点体力难支，心神疲惫地靠在他的身上里，紧紧地拽住他的手臂，想借用他身体的温暖，驱去我心中的那股寒意。我幽怨地低声道，“可是，我好担心……”

冥向天什么也不说，反手把我抱得紧紧地，不停用手拍着我的后背。

当他把我圈进怀里时，我那慌乱的心顿时像是找到了依靠一般，渐渐地平静下来。

感觉自己的心情不知道在火锅里煎熬了多久，终于熬到了胡太医出来。

“启禀皇上，帝君已经醒了！”

“真的？”大喜之下，我马上推开了冥向天跑了进去。完全没有看到身后的人，正一脸失落的看着我的背影。

轩辕凛上前拍了拍冥向天的肩膀，“大哥，你别在意！我相信你们当中任何一个人受伤，雪儿都会是这样担心的。”

冥向天的脸上闪过一丝苦涩的笑，“其实我心里明白，但是，这心里总是会情不自禁地感到酸楚。我没事！我们也进去吧！阿飞，你在看什么？姐姐已经进去了！”

“啊……姐姐走了，我要找姐姐，我要去找姐姐……”阿飞赶紧蹦跳着往房内冲了进去。

冥向天和轩辕凛彼此相视一眼，摇了摇头，苦笑一下，也赶紧跟着走了进去。

进屋一看，我正趴在焰的身上，激动地哭得“稀里哗啦”的，一边哭，我还一边带着心疼地责骂着他，“你怎么那么笨？明明可以赢的，还偏偏落了下风，又故意让自己受伤。就算我们真的赢了他，他金海凌又能怎么样？他要打，大不了我们奉陪就是了嘛！看到你这样受伤，你知道我心里有多难受吗？呜……”

焰的脸色现在稍为好了一点点，虽然看起来还是有些虚弱气喘，但至少不像刚才的惨白色那么吓人。

他伸出大手，温柔地将我散落在前面的长发别到后面去，仍然是那么淡雅温柔地对我笑着，“雪儿，对不起！让你担心我。乖！别哭了，再哭下去，我这心都要碎了！心碎了，可就真的没法治了哦！乖……不哭啊……何况，现在我这不是好好的没事吗？如果这样能让金海凌马上回国，那我受伤也是值得的呀！他多留一天，我们就多担心一天。你们几个说，是不是？”

“是啊！雪儿，大哥这么做，也是想快点解决他这个大麻烦哪！既要输，又不能输得太难看。所以，大哥才拼着自己受伤，也要圆满的让这个金海凌险胜了这一局。”冥向天也附和着解释。

“好了！我不生气就是了！记着，下不为例！”我娇嗔地瞪了他一眼。

“嗯……都听娘子的。”焰笑眯眯地应着。

“皇上，微臣还有话要禀。”胡太医此时在一边插嘴说。

“说吧！”

“帝君的伤除了服用臣开的药方外，晚上的时候，最好有个内力深厚的人，可以用内力帮助帝君疗伤，这样的话，帝君会好得更快一些，大概三天就可以和常人无异，但是要恢复功力，怕是至少要半个月以上。”

听到胡太医的话，我这才拍了拍脑袋，我怎么会忘记这个办法呢？自己可真是榆木脑袋，笨死了！一边想一边忍不住咒骂出声，“唉啊！真是笨！”

胡太医站在一边，赶紧惶恐地跪了下来，“微臣愚笨，请皇上恕罪！请皇上恕罪！”

“咦？胡太医，你跪下干什么？”我惊讶，“朕骂的是自己，不是骂你，你快起来！快起来！”

“谢皇上恩典！”胡太医这才站了起来，抹了一把额上的冷汗。这皇上，说话也不说清楚，吓死他了！

“胡太医，现在帝君可不可以移动？”我笑着问胡太医。只要焰没事就好，心情也随着他的醒转而变得好了起来。

胡太医恭敬地回答，“只要手脚轻一点，移动是可以的。”

“好，那你去叫部车来，我带帝君回雪颜宫。”

“微臣遵旨！”

*****

没有想到，这焰的苦肉计还真的奏效了。

第二天一早，金海凌就来找我，我见了他，当然是冷冷淡淡，没给他好脸色看。

他的眼里闪过一丝尴尬，带着一丝愧疚地对我说，“本王很抱歉，不慎造成帝君受此重伤，愧对于皇上和帝君，自感无颜再在此呆下去，故而向皇上辞行，本王准备今日辰时即起程回国。”

“此时王爷不必过于介怀！比武难免会有损伤。不过，既然王爷要走，那朕也不便挽留，朕会吩咐人马上安排。”听到他要离开，我的心里终于有了一丝欣喜，焰的伤总算是没有白受。

“本王回国以后，会说服皇上，对雪月国永不出兵！两国建立邦交，这也算不辜负女王陛下的大仁大爱之心了。”这一番话，金海凌说起来倒显得有几分诚意，俊脸上的傲气也收敛了几分，只是眉宇间似是多了一丝忧郁。

“那就麻烦王爷了。朕谢谢王爷的厚爱和支持！”我朝他轻轻福了福身。

其实我身为一国之君，没有必要对他一个王爷行礼，但是，也算是我对他的一份歉意吧！不管他开始来此的目的是什么？只要他没有实际伤害到我们的利益，我便不会再与他计较。

送走了这难缠的金海凌，我顿时感觉一身轻松。心神一放松，整个人也显得神采飞扬起来。

*****

一个月后。

这一个月倒是过得平平静静的，在小忙之余，与几位夫君过着甜甜蜜蜜的日子。

焰的伤势半个月前就完全好了，功力也已恢复。冥向天我已经把他调回了京城，他的大将军之职现由裴振宇接任。

各城七月举行的城试，也已经考试完毕，各城已将上了榜将要赴京考试的人员名单呈报了上来，现在就等这八月京试的到来了，看来到时选拔一批人才出来，可又有得忙了。

而这一个月，我也没闲着，为了提高雪月国的国防能力，改造冷兵器的时代，我想起了打仗时最需要的一样东西，那就是---火药。

在现代，火药称为中国的四大发明之一，我也知道，火药是由硝、硫磺和木炭的混合物制成。

在冥向天和裴振宇交接完以后，我就让他去找这三样东西，并且要求他一定要查明这三样东西，在雪月国的哪个地方产量最丰富，以备后用。

开始他们并不明白我要这些东西干什么用，当我让人把这几样东西混合在一起，找了十个小手榴弹大小的空心小铁弹，让他们把制好的火药放在了里面压实，再接上火引。

做好了这将近十个的手榴弹以后，我便带着焰他们几个到了郊外的小山坡上，把这个手榴弹的威力演示给他们看。

当他们看到一个小小的小铁弹装上火药后，爆炸后的威力竟然可以炸出一个不小的坑以后，个个目瞪口呆，不敢置信地看着，啧啧称奇。

十个小手榴弹，竟然有四个是臭弹，应该是工艺未完善的缘故吧！不过，第一次做，有这个成绩我已经很满意了。

放完后，看着这大大小小的六个坑，他们良久才从震惊中回过神来。几个男人带着一脸的崇拜看着我，冥向天更是兴奋地抓紧了我的手，“雪儿，你真是天才！你是怎么想出这个办法的？”

“嗯。是我做梦梦到的。”我狡黠地笑着。怎么可能告诉他，我是带有现代的记忆呢！

“向天，你说，如果我们把这些小铁弹再做大一倍、两倍、甚至三倍五倍，投放到军营打仗的时候使用，你说，到时我们还需要怕谁？只要他们来攻城，定叫他们有来无回。”

我乐呵呵地娇笑着，想到可以把侵略我们的人打得头破血流，我就笑得更是大声。

笑完之后，我又着重地对仍在激动不已的冥向天说，“向天，这件事就交给你了，回去再让他们继续研制。这个项目一定要所有人严守秘密，把它作为我们的秘密武器。待研制成功以后，再秘密训练一批人学会用它。真有必要时，我们就利用这个炸弹给敌人迎头痛击，在战争时可以快速地抢占先机，打起来也必定可以减少战争带来的人员伤亡。”

我虽然没有雄霸天下的野心，但怎知别人会没有？这个安南国的皇上始终是不安分的人，如果他还要再打，我是绝对不会客气的。

“你放心！我一定会做好的！”冥向天朝我行了个军礼，信心百倍地保证着。

这件事告一段落后，在行政上，我也开始正式着手成立廉政公署，我希望雪月国的廉政公署，可以像现代的香港廉政公署一样，真正可以做到廉洁奉公，不贪一丝一毫。而这个署长的人选，正是由陆青担任，由他直接向我汇报负责。

所以，看似清闲的一个月，其实倒也还做了不少事情。

*****

这天，刚下完朝，我又呵欠连天，最近怎么这么瞌睡啊？好像怀孕的人似的！想到这个可能，我突然吓得睁大了眼。

怀孕？不会真的中奖了吧？想起这段时间，经常跟他们一起，又没做避孕措施，那还真是非常、非常、非常的有可能耶！

我的眉头皱了起来，脑筋高速急转，仔细想着上个月的大姨妈是什么时候来的？好像是迟了有那么一段时间了，真的中招了？怎么办？呜……这辰星宝贝才几个月大，还没脱奶，我这还没松几口气，又来了？

“啊……”我被这个感觉有点可怕的消息，气得抱头痛苦大叫。

我的突然失态，吓得跟在我身后的小蝶和小舞大惊，“皇上，你怎么了？”

“小蝶，你马上去太医馆请胡太医过来，就说朕身体不舒服。快去！”

我则急忙回了寝宫，躺在床上等着胡太医的到来。

经过胡太医的一番望闻问切，最后毫不留情的给我判了死刑，“恭喜皇上，贺喜皇上，皇上有喜了！”

“胡太医，你确定没诊断错误？”我抱着侥幸又问了一遍。

“微臣敢保证，绝对不会错！皇上，微臣已经仔细检查两遍了。”胡太医恭敬地回答。

“好了！你下去吧！这件事你先不要说出去。”我无力地挥了挥手。

我懒懒地躺在床上，闭上眼，思绪却停也停不下来。

又有了？那这个孩子的爹，会是谁呢？到时别又像辰星一样，弄不清楚宝宝他爹是哪位？那可就大条了！

正在我烦得不行的时候，那几个罪魁祸首争先恐后地都冲到我的面前来，齐齐抢着问，“雪儿，你怎么了？怎么了？是不是哪里不舒服？太医怎么说？”

几个人的问候声，吵得我更加烦，都是这几个祸害，呜……我咋这么命苦哩！下回一定要注意，不能再上当了。
