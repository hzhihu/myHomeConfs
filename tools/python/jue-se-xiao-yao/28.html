第一卷 烟雨濛濛 第二十七章 冷汉柔情

就在我和冥向天亲密地搂在一起，吻个没完，快擦枪走火的时候，大门被“嘭”的一声打开了。吓得我们赶紧松开了唇放开身子，齐齐掉头向门口张望着，正好对上风沁阳和风儒尔父子那双惊骇得圆睁着的眼睛，他们正用不敢置信的表情看着仍然搂在一起的我和冥向天。

风沁阳一脸痛楚，眼冒火花，那熊熊燃烧火焰似是要将我们烧死似的。稍倾，风儒尔便暴怒地大喝一声：“雪儿，你们在干什么？”

在风儒尔的暴喝下，吓得我的身子抖瑟了一下，我心急地想起身，却感觉到腰上的那只大手搂得更紧了。我不解地看向冥向天，他看向我的眼神深邃明亮，朝我肯定地点点头，像是示意让我把一切都交给他处理。

我惊慌的心马上便安定了，乖乖地把头垂了下去。既然他要做英雄，小女子又怎么能剥夺他的权利呢？我就乖乖的做他怀里的小舵鸟好了。

只听冥向天的声音在我头顶响起，“风丞相，你放心！既然我和小雪有了肌肤之亲，我会娶雪儿的，皇上那里自有我会给他交待，我会让他撤掉封妃圣旨，你就放心吧！现在，你们能不能出去一下？我和雪儿商量商量再去找您，行吗？”

呵呵，这是我认识他以来第一次听他这么长篇大论的说话哩！而且，他对他们说话的声音可不像对我，特别冷冽，连在他温暖的怀里依附着的我都能明显感到一股寒意上身，我偷偷的抬眼看了他一下，发现他那双凤目正寒光四射，像是一束束慑人的闪电直指人心，恐怕老爹和二哥现在的感觉并不好受吧？

“冥将军，事已至此，你当然得给我们一个交待。好，我们就先行退下！沁阳，我们走！”风儒尔的声音依旧沉稳，却也有一种隐晦的强势，怎么着他也是一个丞相，可不是普通人，怎么可能被冥向天三言两语就打发了就吓倒了呢？

“爹，那雪儿……”风沁阳看着还窝在别的男人怀里的我，他又不敢上去把我扯出来，只好向风儒尔求救了。

“雪儿，你还未出阁，有些事不管你愿不愿意，但是女人家该守的礼还是要守的，不要丢了我们风家的脸面。”爹最后一句话说得很重很重，让我委屈得差点又掉下泪来，无助得如同溺水的人，紧紧地抱住了怀里的这根救命浮萍。

“好了。他们走了。”冥向天拍着我的背，声音仍是冷冷的，却渗出一股难得的温柔。

“将军，你会疼我吗？”我抬起有点发红的眼睛，带着点受伤的小动物般的眼神，用可怜兮兮的、有点无助、有点软弱、有点期盼地语气问着他。

他大手轻抚着我的一头青丝，手指缠绕着发丝，“把一切交给我，我既然答应了，就会处理好的，你别担心。你嫁给了我，我当然会疼你的！”为了表示肯定，他还轻轻地咬了一下我的耳垂，又在我耳边低语了一句，“以后不要再叫我将军，叫我向天。”

我满怀感激地看着他，娇羞地点着头，“嗯”了一声后，乖乖地看着他娇滴滴地叫了一句“向天……”便窝入了他的怀里。哪怕只是这一刻的贪恋也好，以后会怎么样，我无法预知，只有等以后再说。但是在这一刻，我愿意把身心都交给这个愿意担当并给了我温暖的男人！

两个人就这样静静的依偎着，这一刻，很宁静！依偎在这个男人的怀里，我也觉得特别安心，像是为自己找到了一个依靠。

“向天，你有什么办法让皇上撤掉圣旨？”我依着冥向天，好奇地问。

“山人自有妙计。”他不肯说。

“告诉我嘛！”我撒着娇，轻喊他的名：“向天……”酥声入骨，更是让他愈发的疯狂。

他快速地脱掉自己身上的束缚，两具身躯开始紧紧纠缠，我已经头晕目眩，脑子发热得无法思考，突然听见外面传来一声大喊，“皇上驾到！”

这一声大喊，如同冰水瞬间浇息了我们体内沸腾的火苗。

冥向天低咒一声，把头埋在我的颈侧，抱怨着，“要多来几次，我不得那不举之症才怪！”

“向天，向天，你快起来！要是被皇上看见了我们这样，那可就完了！你快点穿衣服！”我着急地想推开他，但是，他却慢条斯理地动也不动，大手还在我滑如凝脂的皮肤上流恋着，嘴里还在喃喃地说，“你的皮肤就像那顶级云缎，柔软细滑又有弹性，摸起来可真舒服。小妖精，真想现在就要了你。可惜……这皇上怎么偏偏拣这个节骨眼上驾临呢？”满嘴地抱怨。

我就不知道冥向天面对皇上的到来咋也这么镇定呢？别的人一听皇上来，还不得吓得屁滚尿流、手慌脚乱的，可是他呢，俊脸上一脸的不以为然，完全不当皇上一回事似的，还不让我起身，难道他和皇上之间还有什么秘密？还是说他兵权在握，皇上对他有所畏惧？

不管怎么样，我还是想起身穿衣，要是被人家进来看见这一地散乱的衣衫，那还不得给气晕了，那我的名节、皇上的面子可都顾不住了，“向天，快点……万一皇上来了，看见我们这样，那就完了。”我急得快哭出来了。大爷，你有倚仗，小女子我可没有哇！这可是掉脑袋的事，怎么着皇上的面子还是要顾的吧？

看我都快急哭了，冥向天这家伙才咧着唇角放过我。

我赶紧下床，也顾不得什么羞不羞的了，三下二下当着冥向天的面捡起散落在地下的衣服穿了起来。待穿好了整了整看没什么不妥了，这才算是松了一口气，回过头，正对上冥向天似笑非笑的看着我，我就是脸皮再厚，这时也忍不住脸色通红，忍不住娇嗔地瞪了他一眼。想起刚才差一点就被他吃了，要是他现在就知道我不是完壁之身了，还肯要我吗？真是险！这皇上还是来对了。

我转过头看了看冥向天，看他还是一副衣衫半露的惑人模样，赶紧上前帮他也系好衣服，柔顺的发丝轻垂在他的脸上，让他眯了眯眼，用手撩起我的长发放到鼻间闻了闻，冲我露出一个痞子似的坏笑，“嗯……真香……”看到我羞窘得通红的脸，他又开心地哈哈大笑起来。

这时，大门再一次打开了。

轩辕翌带着一身萧瑟的冷意出现在了门口，俊脸上一副高深莫测的表情，刀锋似的眼神在我和冥向天之间转了转，我心里一寒，连忙跪下拜见，“民女磕见皇上，皇上吉祥！”

冥向天也要挣扎着起身下来拜见，轩辕翌却阻止了，“冥将军有伤在身，就免礼了！你们都退下吧！朕有话和冥将军说。”

众人退下，我也低着头起身往外走去，只要这轩辕翌一在场，我就觉得压抑得喘不过气来。

结果，在经过轩辕翌身边时，手臂却被一只如铁钳般的大手给抓住了，“雪儿，你也留下！”
