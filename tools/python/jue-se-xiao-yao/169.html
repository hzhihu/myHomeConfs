后续卷 第五章 小猴子闯死亡谷

无尽黑夜。在暴雨滂沱中，有几条身影显得狠狈地、一路跌跌撞撞地向前狂奔着，终于，在他们感觉气息快要耗尽之时，来到了一个在雪月国与澜沧国交界的小镇上，这个小镇的名很普通，叫“大柳镇”，几位黑衣人左看右看发现无人跟踪之后，几条身影便没入一间农舍之内。

农舍的屋内，只有一盏昏暗的油灯在闪烁着，为这黑沉的夜照出一丝亮光来。

摆放油灯的桌前，此时正坐着一位长须的黑衣老者，老者闭着眼睛，那张枯黄的脸如同木蜡一般的毫无表情，整个人身上散发出一种死气，与屋外的闪电雷鸣相互交错着，在这不寻常的深夜让人看来，更觉阴森恐怖。

屋外的几位黑衣人进入后，看到端坐在桌前的黑衣人，似是极为惧怕，未等他开口，便马上跪在了老者的面前，低下头微颤着声音向黑衣老者回报着，“回禀左护法，属下等无能，此次到峡谷伏击的[和]字辈兄弟，除我们几个逃出生天以外，其余已全部遇难。请左护法惩罚！”

黑衣老者在听到他的回报后，双目顿时张开，散出一股凌厉骇人的光芒，随即便阴恻着声音盯着他说，“哼！你们确实无能！出动了这么多人，竟然还摆不平他们，真是废物！罢了，先给我说说经过！迟些再对你们作出处置！”

“是！今天属下们按线报提供的时间，一早就已在一线天峡谷布置好了陷阱。时间一到，果然看到了雪月国皇子的驾鸾经过，可没有想到，他们在进入一线天峡谷时，突然停了下来，致使我们预先布置的陷阱全没有用上。”

“之后，眼见等到天黑了，属下们没有办法，只有吩咐众人杀了上去。没有想到，他们当中突然飘飞出一位美如天仙的女人，冲着我们娇媚的笑了笑，大家都被她的美迷倒的时候，还未反应过来，她已经开始杀了起来。她的武功太高了！太可怕了！实在是太可怕了！”

想起当时那位绝色美人下手毫不留情，所过之处，如入无人之境时的勇猛，他就硬生生地打了个冷战，微颤着声音又继续说，“她每出一剑，必会倒下一位兄弟，我们所有人合起来，也不是她的对手，更可怕的是，像她这样的高手，竟然不止她一个。后来，又出来二位功力绝高的男人，在短短的一刻钟，我们近百名的兄弟，便倒下了三分二。我看情形不对，便率着这几位兄弟杀了出来，赶紧来面见护法回禀情况，属下无能，请护法处置！”

黑衣老者的眼里闪出一丝兴味，“哦！竟然有这样厉害的女子？和一，你可知道她是谁人？”

“属下只听到她身边的一个衣着卫士将军服的人，称呼她为皇上！”

“皇上？”黑衣老者惊得“腾”地从凳子上站了起来，后来又像是感觉自己太过于惊讶，马上又坐了下来，喃喃自语着，“难道……她就是传闻中雪月国那武功出神入化的皇上？她的武功真的这么高？”

一刻钟时间，便将百名[和]字辈的兄弟消灭？黑衣老者沉思着，这种高绝的功力，不但连他这位死亡谷的左护法做不到，恐怕连副谷主也无法在一刻钟内将百名[和]字辈的兄弟消灭干净啊！太可怕，确实是太可怕了！

这种功力究竟高到了何种程度？难道她的功力真的像谷主的一样，一出手便足以毁灭一切？高到让人无法仰其项背？让人觉得他是神、而不是人的境界？

“如此看来，那第二拨理伏的人，肯定也是完了！你说，我该怎么向谷主交待？”黑衣老者叹道。

“请护法责罚！”和一低头领罪。

他低头沉思了一会，便说，“事不宜迟，我们得赶紧回谷向谷主报告这件事。走！”

“是！”剩余黑衣人齐齐领命，跟随在他身后，朝着一个方向奔去。

而他们的身后，一直有一只小黄雀紧紧跟随着。

*****

黑衣人最后的目的地是大柳镇背后的后山谷，翻过一座高高的山岭，然后又七弯八拐的避过那些机关阵法，这才进入一颗大石上刻着“死亡谷”三个大字的谷口。

孙天赐化身黄雀紧跟在他们的身后，不由得暗自咂舌，这个地方真的异常隐秘，如果不是他们带路，恐怕还真的是难以找到他们的老巢。再看进入这山谷前的机关阵法，就算被他们知道了，恐怕常人也根本无法到达里面，想要消灭他们那更是不可能。

这个死亡谷到底是谁创立的？他们创立死亡谷的目的到底是什么？他们对母后和皇兄下手，究竟是受人所托？还是他们自己计划的行动？

一切事情尚未明朗，孙天赐感觉到事情现在是越来越有意思了，扑朔迷离得让他感觉到了难得的兴味。

这几年，他和风天恩两个人携手共闯江湖，一直感觉那些江湖中人太没有挑战性了，随便一出手就能打败他们，久而久之，他也没了入江湖的兴趣，还不如在宫里陪着娘亲，在娘面前撒撒娇耍一耍还开心。

可是，这一次，第一次让他有了强敌对手的感觉，也让他有了一种不安的感觉。

紧紧跟随在黑衣护法的身后，随即，谷内便出现一个小镇似的村落，他们走到其中一座四合院中，只听黑衣护法伸手在铜环上敲了二长一短的声音后，门便开了。黑衣人低声对开门的童子道，“烦请兔兄通传，左护法尊虎求见谷主。”

孙天赐仔细一看，心下一惊，这个守门的被黑衣护法称为兔兄的功力，他竟然看不出他的高低来，心里不由得对这个谷主更加好奇，行动却也更加谨慎小心起来。

他先飞上屋顶，从屋顶悄然飞入院内。没有想到，这普通门面的一个四合院，里面竟然如此之大，难道是采用了障眼法的缘故？

一想到这个可能，他心里悚然一惊！难道……会是妖魔鬼怪出来作祟？

为了探明真相，他隐藏起自己的行踪，将小黄雀的身形换成了一只更加难以察觉的小苍蝇，静等着黑衣护法的进入。

果不其然，那名兔兄很快便从堂内出来，宣黑衣护法尊虎入内觐见，其它人则被拦在外面，让他们暂时先回自己队里歇息。

“左护法，谷主在内堂等候，请吧！”那名兔兄打开门请尊虎进入后，自己则守在了门口。

“谢兔兄！”尊虎朝他拱了拱手，便走了进去。

化身为小苍蝇的孙天赐赶紧跟着飞了进去，又跟着尊虎来到了所谓的内堂。

*****

内堂。

左护法尊虎走了进去，内堂悬挂着紫色纱缦，在紫色纱缦的后面，有一软榻，此时，软榻的四周围绕着几名姿色上乘的侍女，而侍女的中间，坐着的便是令孙天赐大为惊诧的谷主，没有想到，她竟然是一个妖艳绝美的女人！

“属下尊虎拜见谷主！”尊虎竟然跪下叩见，这礼看来比自己参见母后还严哩！

“起来吧！事情办得怎么样了？”娇媚入骨的声音，软得几乎让人的骨头都软了。

“是属下无能，请谷主责罚！”尊虎低头不敢再吭声。

“你的意思是，事情搞砸了？”

又一声淡然缥缈的问话，却让尊虎的身躯震了震，跪在地下不敢起来，只是低低地应了声，“是！”

“那你还回来做什么？回来领罪吗？”

美女谷主一边笑着，那只抚着侍女的纤手突然一用力，便让那名侍女痛得尖叫出声，“呀……谷主饶命哪……谷主饶命哪……”

侍女那痛苦的尖叫声，不但没有即时停下，反而是愈来愈痛苦的在那里尖叫着求饶，像是谷主特地让她叫给尊虎听的。

最后，随着这位美女谷主的一扬手，那侍女的身子便如飞絮一般地飘落，破败的身躯就这样摔落在了尊虎的面前。而侍女那痛苦得死不瞑目的表情，也就这样活生生地显露在尊虎的面前，让他那张本已枯黄的脸色，更加惨白得骇人，冷汗如水珠般地滴落在地上，很快便形成了一滩水印，继续不断地扩散着……

而化身隐在一边的孙天赐，也被这位绝色美女的残忍给吓了一大跳。
